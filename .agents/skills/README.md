# Fabrik skills

This directory contains reusable Agent Skills for Fabrik workflows.

## Available skills

- `fabrik-smithers-run-forensics` — Investigate Smithers runs in Kubernetes using logs, PVC artifacts, Smithers DB, and remote branch/bookmark verification.

## Usage

Point your agent/client to `.agents/skills/` as a local skills directory.

For details, open each skill's `SKILL.md`.
